# AI Code Debugger

A real-time code analysis tool that detects errors and provides AI-powered suggestions using Gemini API.

## Features

- 🛠️ **Code Error Detection**  
- 🤖 **AI-Powered Fixes** (via Gemini API)  
- 📝 **Error Explanations**  
- 🔒 **User Authentication**  
- 📚 **Error History Tracking**  

## Tech Stack

**Frontend**:  
- React.js  
- React Router  
- Context API  

**Backend**:  
- Node.js  
- Express  
- MongoDB  

**AI Service**:  
- Google Gemini API
